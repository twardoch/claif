# Provider Packages

Claif uses a modular provider system to support different LLMs. Here are the official provider packages:

## `claif_cla`

- **Provider**: Anthropic
- **Models**: Claude 3 (Haiku, Sonnet, Opus)
- **Installation**:
  ```bash
  pip install claif_cla
  ```

## `claif_cod`

- **Provider**: OpenAI
- **Models**: Codex
- **Installation**:
  ```bash
  pip install claif_cod
  ```

## `claif_gem`

- **Provider**: Google
- **Models**: Gemini
- **Installation**:
  ```bash
  pip install claif_gem
  ```
